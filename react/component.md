# 개요
리액트를 사용할때는 컴포넌트를 class 또는 함수로 정의할 수 있다.
class로 정의된 컴포넌트는 많은 기능을 제공합니다. 
React 컴포넌트 class를 정의하려면 React.Component를 상속받아야 한다.


## 컴포넌트 생명주기
모든 컴포넌트는 여러 종류의 생명주기 메서드를 가지며, 이 메서드를 오버라이딩하여 특정 시점에 코드가 실행되도록 할 수 있습니다.
